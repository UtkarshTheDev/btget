## [1.0.0] - 2025-11-24

### Added
- Initial public release of `btget`.
- Core BitTorrent client functionality for downloading `.torrent` files.
- Support for connecting to multiple trackers for peer discovery.
- Real-time progress bar display with download speed and peer count.
- `info` command to inspect torrent metadata without downloading.
- Cross-platform support for Windows, macOS, and Linux.
- Packaged for distribution via npm.
- Optimized for performance using Bun's build tools.
- Comprehensive test suite for core functionality.
- Documentation for usage, contribution, and project structure.
